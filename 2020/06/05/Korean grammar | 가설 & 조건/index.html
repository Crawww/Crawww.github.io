Korean Grammar | 가설 & 조건

#거든

접속  동사 어간 / 형용사 어간 / 체언+ 이 / -았 / 었 / 였 / 겠 /시 + 거든

설명  接续词尾 表假定条件 = “如果”、“要是”

주의  终结谓语一般使用表示祈使的命令式或共动式

예문  

  • 값이 비싸거든 사지 맙시다.

    如果价格贵，咱就别买了。

  • 호시 길이 미끄럽거든 지하철을 타거가자.

    如果路滑，我们就坐地铁去吧。

  • 잘못했거든 진심으로 사과해야합니다.

    如果做错了事，就应该真诚地道歉。

  • 일찍 퇴근하시거든 우리 집에 잠깐 들러주세요.

    如果下班早的话，请您到我家来一趟吧。

  • 만일 그것에 가거든 어머니와 그사정을 잘 이야기해 주세요.

    要是到那个地方，请把这件事情好好和母亲解释一下吧。

#거들랑

접속   동사 어간 / 형용사 어간 / 체언+ 이 / -았 / 었 / 였 / 겠 /시 + 거들랑

설명 I  由거든和을랑 (表强调) 组合而成的接续词尾 常用于口语 表假定条件 相当于如果 要是

예문 I  

  • 꽃이 피거들랑 공원에 갑시다.

    如果花开了，我们就去公园吧。

  • 소식이 없거들랑 희소식인 줄 알아요.

    没有消息就是好消息。

  • 중국에 돌아가거들랑 나에게 전화를 해주세요.

    如果回中国的话，请给我打个电话。

  • 문제가 생기거들랑 선생님께 여쭤 보세요.

    如果有问题，请问问老师吧。

  • 그도 갈테니 부탁할 일이 있거들랑 그에게 말하세요.

    他也要去，要是有什么拜托的事，就跟他说吧。

설명 II   表示推理的条件 能与(으)면和거든 通用，常与时制词尾-았/었/였连用

예문 II

  • 구경을 가겠거들랑 빨래준비하세요.

    要去参观的话就快点准备吧。

  • 틀린것을 알았거들랑 반성을 해야지요.

    知道错了就该好好反省。

  • 이론적으로 알았거들랑 살천에 옮겨야 할 것압니다.

    理论上懂了就该付诸实践。

  • 잘못을 깨달았거들랑 제땨에 고쳐야 합나다.

    意识到错误就该及时改正。

#고 나면

접속  동사 어간 + 고 나면

설명  由表示前后顺序的고 나다和表示条件的면 组合而成的句型 常用于口语 表示如果满足前句就会发生后句

예문  

  • 내 얘기를 듣고 나면 이해할 거에요.

    听了我的话你就理解了。

  • 이 음악을 듣고 나면 마음이 편안해질 겁니다.

    听了这首歌之后，心情会变平静的。

  • 이 책을 읽고 나면 생각이 달라질 거에요.

    读完这本书之后，想法会有所改变的。

  • 비가오고 나면 날씨가 좀 서늘해지 겠지요.

    下了雨之后天气会变凉一点吧。

#고 보니 / 고 보면

접속  동사 어간 + 고 보니 / 고 보면

설명  由动词보다和表示条件的니 / 면 组合而成的句型 表示前项动作完成之后，才发现或明白后项

예문  

  • 명철이의 이야기을 듣고 보니 내 가 너무 내 생각만 한 것 같았다.

    听了明哲的一席话，我觉得我太自私了。

  • 알고 보니 사고을 낸 차가 바로 동생 친구의 차였다.

    经了解，发生事故的是弟弟朋友的车。

  • 사실 알고 보면 이 아이는 아주 착한 아이랍니다.

    事实上你会发现这孩子很善良。

  • 진짜 가까게 사귀고 보면 하나도 안 무서워요.

    亲密交往后会觉得一点都不可怕。

  • 오래 접촉하고 보면 상대방 의 우점과 결점이 보인다.

    长期接触后，对方的优缺点会显现出来。

#기만 하면 (되다)

접속  동사 어간/ / 형용사 어간 / 체언 이 + 기만 하면 (되다)

설명  表示只要……就，一……就

예문  

  • 이번 주말 전으로 완성하 기만 하면된다.

    只要这周末之前完成就行了。

  • 8시 전에 도착하기만 하면 문제없다.

    只要八点之前能到就没问题。

  • 이것보다 좀 더 크기만 하면 된다.

    只要比这个稍微大一点就行了。

  • 이 시게는 건드리기만 하면 열린다.

    这个表只要一碰就会开。

  • 한국어 학과 촐업생이기만 하면 된다.

    只要是韩语专业毕业生就可以。

#는 날엔

접속  동사 어간 + 는 닐엔

설명  相当于 “有朝一日”、“一旦”

예문  

  • 제가 성공하는 날엔 교수남의 은혜를 꾹 갚을 거에요.

    有朝一日我成功了，一定会报答教授的恩惠。

  • 다시 나쁜 짓 하는 날엔 너를 가만하지 않겠다.

    如果你再干什么坏事，我绝不放过你。

  • 사실이 밝혀지는 날엔 어떻게 뒬 각오하세요.

    如果有一天真相大白，会有什么样的下场，请你做好思想准备。

#ㄴ/은/는 한

접속  동사 어간 (ㄴ/은) / 형용사 어간 (는) + 한

설명  表示前句是后句的条件 = “只要……就”

예문  

  • 가격이 높은 한 잘 안 팔려요.

    只要价格高就不好卖。

  • 내가 살아있는 한 포기하지 않겠습니다.

    只要我活着，我就不会放弃。

  • 남의 잘못을 용서해 주지 않는 한 자기 마음도 편할 수가없어요.

    如果不原谅别人的错误，自己心里也不好受。

  • 남복 통일이 되지 않는 한 진정한 의미에서의 민족의 발전은 기대할 수 없어요.

    只要朝鲜半岛不统一，就无法期待真正意义上的民族发展。

#더라먄

접속  동사 어간 / 형용사 어간 / 이(다) / -았/었/였 + 더라면

설명  表示对过去已经发生的事情进行与现实相反的假设，带有后悔、遗憾、庆幸或感叹的语气。后文的内容与现实相反 = “如果当初……的话，那么…..”

예문  

  • 집에서 좀 일찍 떠났더라면 기차를 안 놓쳤을걸.

    如果早点从家里出发，就不会错过火车了。

  • 그때 서두르지 않았더라면, 우리는 기차를 놓치고 말았을거예요.

    那个时候要是不快点的话，我们就赶不上火车了。

  • 주말에 좀 쉬었더라면 몸살은 안 났을텐데.

    周末如果休息好了的话，就不会积劳成疾了。

  • 좀 더 조심히 했더라면 망신을 당하지 않났을겁니다.

    如果当初小心一点的话，就不会丢人现眼了。

#던들

접속  동사 어간 / 형용사 어간 / 이(다) / -았/었/였 + 던들

설명  接续词尾 对过去的事情进行假设 常用于口语中 = “如果当时……的话” = “더라면”

예문  

  • 서로 조금만 양보했던들 저렇게 크 사고는 나지 않났을 걸예요.

    假如当时互相让步一点，就不会发生这么大的事故了。

  • 술을 더 절제했던들 이 지경에까지는 이르지 않났을 걸.

    如果当初喝酒节制一点，就不会到这个地步了。

  • 조금만 더 건강했던들 이만한 일에 쓰러지지는 않았을 텐데.

    如果当时身体再健康一点，就不会因为这点事情倒下了。

#만 아니면

접속   체언 + 만 아니면

설명  表限定的“만” 表否定的“아니다” 表假设的“면”组合而成的句型。表示限定条件假设。 = “只要不”

예문  

  • 경멸만 아니면 모든 것을 참아낼 수 있다.

    只要不是蔑视，其它什么都能接受。

  • 밤 늦은 시간만 아니면 언제고 전화해도 괜찮아요.

    只要不是三更半夜，随时都可以打电话。

  • 원룸만 아니면 됩니다.

    只要不是单人间就可以。

  • 애들 공부만 아니면 시골에 가서 살고 싶어요.

    要不是为了孩子们，我真想去乡下生活。

# (ㄴ/는) 다면 / (이) 라면

접속  동사 어간 / 형용사 어간 / 았 / 었 / 였 / 겠 + (ㄴ/는) + 다면 & 체언 + (이) 라면

설명  接续词尾 表假定条件 = “如果说……那么”

예문  

  • 네가간다면 나도 같이 갈 거야.

    你去的话我也一起去。

  • 내가 너처럼 예쁘다면 좋겠다.

    我要是跟你一样漂亮的话就好了。

  • 꼭 염섹이나 파마를 하셔야겠다면 두피와 모발에 자극이 적은 약물을 선택하네요.

    一定要染发或者烫发的话，选对头皮和头发刺激小的药水吧。

  • 내가 너라면 그런 일은 절대 하지 않는다.

    我是你的话绝对不会做出那样的事情。

  • 네가 도와 주지 않았다면 나 혼자너 혼났을거야.

    要是没有你的帮忙，我一个人肯定吃不消啊。

# (ㄴ/는) 다면야

접속  동사 어간 / 형용사 어간 / -았 / 었 / 였 / 겠 + (ㄴ/는) 다면야

설명  表示如果满足前项条件，自然出现后项结果。常用于口语中 这里“야”表示强调的语气

예문  

  • 윈인만 찾아낸다면야 이 문제는 완전히 해결할 수 있습니다.

    只要找到原因，这个问题就能彻底解决了。

  • 잎차와 뒤차의 안전거리만 유리한다면야 사고를 줄일 수 있을 텐데요.

    如果保持前后车的安全距离，就能减少事故。

  • 가방을 살무실에 두었다면야 있을 텐데, 없는 걸 보니 다른 데에 둔 것 같아요.

    如果把包放在办公室就应该有啊，没有的话可能是放在别的地方了。

# (으) 면야

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 + (으) 면야

설명  对过去的事情进行与现实相反的假设，常用于口语中。 这里“야”表示强调的语气

예문  

  • 기차를 탔으면야 돈도 절약되고 얼마나 좋겠어요.

    当初要是搭火车的话还能省钱，多好啊。

  • 대우가 좋았으면야 제가 왜 그만두었겠어요?

    如果当时待遇好的话，我为什么还要辞职呢。

  • 열심히 공부했으면야 좋은 대학교에 들어갈 수 있었을 거예요.

    如果当初用心读书，也许就能进好大学了。

  • 그 때 대학생이였으면야 좋은 대상을 만날 수 있었지.

    如果当时是大学生的话，能找到很好的对象。

# (으) ㄴ/ㄹ 줄 알았더라면

접속  동사 어간 / 형용사 어간 / 이 + (으) ㄴ/ㄹ 줄 알았더라면

설명  带有后悔的语气 = “如果早知道…那么”

예문  

  • 야구시합이 그렇게 제미있을 줄 알았더라면 나도 갔겠는걸.

    如果当时知道棒球这么有趣的话我也去了。

  • 그때 그것을 사랑인줄 알았더라면 너를 보내지 않았을 것이다.

    如果当初知道那时爱，我也许就不会放开你。

  • 이렇게 추울 줄 알았더라면 겨울 웃을 준비해 둘걸.

    早知道这么冷，就多准备点过冬的衣服了。

  • 비가 올 줄 알았더라면 우산을 가지고 떠났을 텐데.

    早知道下雨，出门就带雨伞了。

# (이) 라야 (만)

접속    명사+ (이) 라야 (만)

설명 I  表示条件的接续词尾。表示前项为后项的必要条件。相当于只有……才 否定形式为아니라야(만)

예문 I  

  • 이 일은 자네라야 감당할 수 있을걸세.

    这件事只有你才能胜任。

  • 공사가 끝난 뒤라야 내금을 지불할 수 있다.

    工程结束之后才能付款。

  • 주말이라야면 하루 휴식할 수 있다.

    只有到周末才能休息一整天。

  • 친자식이 아니라야만 증인자격이 있다.

    只有非亲子关系才能做证人。

  • 우리 나라는 유월 장마라야 풍년이 든납니다.

    我们国家只有在六月份进入雨季才能获得丰收。

설명 II   表示对指定的某一件事进行说明。

예문 II

  • 치마라야 입고있는 이것 말고는 없다.

    裙子除了身上穿的这条之外，再没有其他了。

  • 자방 미인대회라야 그 규모가 크지 않다.

    地方选美大会的规模不大。

  • 할아버지 재산이라야 집 한 채가 전부다.

    爷爷的全部财产也就是一套房子。
