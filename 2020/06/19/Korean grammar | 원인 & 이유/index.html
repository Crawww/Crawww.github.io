# 고 해서

접속  동사 어간 / 형용사 어간 / 이 + 고 해서

설명  表示导致某事发生的多种原因中，只列举其中一个，即除了提到的原因外还有其他原因 常用 “도 - 고해서“

예문  

  • 날씨가 따뜻해지고 해서 아침마다 산책을 가요.

    天也暖和了，所以每天早上都去散步。

  • 주변도 조용하고 해서 그 동네로 이사를 했어요.

    周边很安静，所以就搬到那个小区了。

  • 장학금을 받고 해서 친구들과 한잔했어요.

    得了奖学金，所以和朋友们出去喝了一杯。

  • 기분도 울적하고 해서 산책을 나가기로 했다.

    心情也有点郁闷，所以决定出去走走。

  • 이러다 죽는 거 아닌가 하고 겁도 나고 해서 그를 꼭 껴안았다.

    照这样下去岂不是会死，觉得害怕所以紧紧抱住了他。

# 기 때문에

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 / 겠 + 기 때문에

설명  表示原因 = “因为，所以”

주의  表示原因或理由的内容出现在谓语部分时，往往以기 때문이다的形式结尾，如句4。

예문  

  • 주말이기 때문에 백화점에 사람이 엄청 많아요.

    因为是周末，所以商场的人很多。

  • 은정 씨는 어제 밤새 공부기 때문에 오늘 늦게 일어났어요.

    恩静昨晚熬夜学习，所以今天起来得很晚。

  • 평수에 아주 바쁘기 때문에 친구들과 자주 연락을 못합니다.

    因为平时很忙，所以不能经常和朋友联络。

  • 현대 사회가 하루가 다르게 변화하고 있어 ‘빨리빨리’ 적응하는 것이 성공에 유리하기 때문이다.

    那是因为现代社会发生日新月异的变化，快速适应有利于成功。

# 기에

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 + 기에

설명  接续词尾。表示原因或依据 = “因为……”

예문  

  • 반가운 손님이 오셨기에 버선발로 달려 나갔다.

    因为来了贵客，所以连鞋子都没有穿，急忙跑出去迎接客人了。

  • 전번 조말에는 어디로 가셨기에 전회도 안 통해요?

    上周末您去了哪里了，怎么电话打不通？

  • 도대체 얼마나 울었기에 눈이 퉁퉁 부었을까?

    到底哭了多久，眼睛会肿成那样？

  • 학생들이 두대체 무슨 죄를 지었기에 저런 징벌을 받아야 하는가?

    学生们到底犯了什么罪，要受到那样的惩罚？

# 길래

접속  동사 어간 / 형용사 어간 / 이 (다) / -았 / 었 / 였 + 길래

설명  表示原因，常用于口语中 “기에” 的口语型

예문  

  • 배가 너무 고프길래 라면을 끓여 먹었어요.

    肚子太饿了，所以煮了泡面吃。

  • 얼마나 그 사람이 좋길래 그렇게 정신을 못 차리냐?

    那个人是有多好，把你迷得这样神魂颠倒。

  • 아무리 약을 먹어도 안 길래 병원에 기봤다.

    再怎么吃药也没效果，所以去了医院。

# 느라 (고)

접속   동사 어간 + 느라(고)

설명 I  接续词尾。表示原因。前后句子的主语一致，往往省略后句主语。 느라고 可以缩写为 느고

예문 I  

  • 자느라고 도둑이 들어온 걸 몰랐어요.

    睡着了，所以不知道小偷进来了。

  • 공부하느라고 다크서클이 생겼어요.

    因为学习，出现了黑眼圈。

  • 저희 부무님은 농장을 돌보시느라고 바쁘십니다.

    我的父母因为照料农场，所以很忙。

  • 그동안 어머니를 모시느라 고생 많았서.

    一直以来侍奉母亲，辛苦你了。

설명 II   接续词尾。表示目的。前后句子的主语一致，往往省略后句主语。 느라고 可以缩写为 느고

예문 II

  • 은지는 옷음을 참느라고 딴 데를 보았어요.

    恩知为了忍住不笑，看了别的地方。

  • 방을 구하느라 뛰어다녀요.

    为了找房子到处奔走。

  • 그녀는 눈물을 감추느라 한참 눈을 감고 있었다.

    她为了忍住泪水，闭了好一会眼睛。

  • 영화를 보느라 점심을 긂었다.

    为了看电影，没有吃午饭。

# 는 바람에

접속  동사 어간 + 는 바람에

설명  表示因为某事而导致意想不到的结果，这一结果往往是消极的，负面的。

예문  

  • 급히 먹는 바람에 체햤다.

    因为吃太急，所以噎着了。

  • 어제 눈이 오는 바람에 길이 미끄러웠다.

    因为昨天下雪，所以路面很滑。

  • 아이는 배탈이 나는 바람에 학교에 결석했다.

    因为孩子吃坏了肚子，所以没去学校。

# 는 통에

접속  동사 어간 / 형용사 어간 + 는 통에

설명  表示原因或根据，带有很强的责怪或不满的语气。

예문  

  • 아이들이 떠드는 통에 잠을 잘 수가 없어요.

    孩子们太吵了，没法睡觉。

  • 급하게 나오는 통에 휴대폰을 잊고 나왔어요.

    出来得太急，所以出来时候把手机落下了。

  • 담당자가 없는 통에 업무가 영 뒤죽박죽이 뒤고 말았어요.

    因为没有负责人，业务变得乱七八糟。

# 다 (가) 보니 (까)

접속  동사 어간 / 형용사 어간 / 이 + 다(가) 보니(까)

설명  引出结果。表示长期处于某种状态，自然引发出另外一个结果。这是由 “다가 (表示原先动作发生变化或停顿)”和“보니까 (表示原因、理由)”组合而成的句型。“보니까”也可以说成“보니”

예문  

  • 오래 살다 부니까 별꼴을 다 보겠네.

    人活久了，什么事都能遇见。

  • 너랑 즐겁게 놀다 보니까 걱정이 다 날아간 거 같아.

    跟你玩得好开心，我的烦恼好像都消失了。

  • 매일 한국 친구를 만나다 부니까 한국어를 잘 하게 됐어요.

    天天见韩国朋友，自然韩语越来越流利了。

  • 저의 친구들은 다 남자 친구가 있는데 제가 좀 똥똥한 편이다 보니까 님자친구도 없어요.

    我的朋友们都有男朋友，但是我因为胖，还没有男朋友。

  • 연못을 들여다 보니까 과연 물속에 황금의 그림자가 비치고 있었다.

    往莲花池里一看，果然看到水里映出金色的影子。

  • 워낙 바쁘다 보니까 다들 생각하지 못했나 봐요.

    可能是因为大家都很忙，所以没想到吧。

  • 돈을 너무 아끼다 보니 인생을 즐기지 뭇했어요.

    太节省了，结果都没能享受人生。

  • 친구의 재미있는 이야기를 듣다 보니 시간 가는 줄 물랐네요.

    听朋友讲有趣的故事，结果都不知道时间过得这么快。

  • 해외에 나갈 기회가 없다 보니 우물 안 개구리가 되었네요.

    一直没有机会出国，结果都成了井底之蛙。

# 더니

접속  동사 어간 / 형용사 어간 / 았/었/였 + 더니

설명 I  表示原因。前项往往是说话者亲身经历或耳闻目睹的事情，而这一事情正是导致后面结果的唯一原因。前后句的主语要一致，后面的结果是原先意想不到的。

예문 I  

  • 운동을 하였더니 온몸니 쑤신다.

    运动之后，全身酸痛。

  • 피아노 연습을 열심히 하더니 이젠 퍽 잘 히는군요.

    看你认真练了一段时间钢琴，现在弹得非常好。

  • 며칠 잠을 못 잤더니 좀 피곤하네요.

    几天没睡，有点疲劳。

  • 어제 술을 많이 마셨더니 속이 쓰려서 이침밥을 못 먹겠어요.

    昨天喝了好多酒，现在肚子难受，吃不下早饭。

설명 II  表示转折。前句表示说话者亲身经历的事情，或某种现象发生，后句表示有与其相反的新、情况或新变化发生。后面接的是意想不到的结果。

예문 II  

  • 어제는 기운이 없더니 오늘은 기운이 넘치는구나.

    昨天无精打采，今天浑身劲儿。

  • 어렸을 때 늘 올더니 아주 어른스러워졌어요.

    小时候那么爱哭，现在成熟多了。

  • 물여 빠진 놈 건져 줬더니 내 보따리 내놔라 한다.

    救了落水的家伙，可他却让我交出包袱（恩将仇报）。

  • 어제는 덥더니 오늘은 시원하다.

    昨天很热，今天却很凉快。

설명 III 表示递进。

예문 III

  • 바람이 불더니 이제 비까지 오는군요.

    刚才刮风，现在还下起雨了。

  • 얼굴이 예쁘더니 마음까지 곱네요.

    人美，连心也美。

설명 IV 表示继续出现新情况。

예문 IV

  • 그녀가 바짝 다가서더니 귓속말을 소곤거렸다.

    她靠紧我，在我耳边小声低语了几句。

  • 어둠 속어서 한 여자가 나타나더니 재빠른 동삭으로 어느 집으로 들어갔다.

    黑暗中出现了一名女子，飞快地跑进了一户人家。

  • 우뢰소리가 들리더니 금방 큰 비가 쏟아져내렸다.

    刚听到雷声，就下起了雨。

# 덕분에 / 덕분으로

접속  동사 어간 / 형용사 어간 ㄴ/는/던 // 체언(의) + 덕분에/덕분으로

설명  表示对受恩的感激之情 = “多亏，托….福”

예문  

  • 덕분에 목숨을 건졌습니다.

    多亏您，获救了。

  • 여러분 덕분에 임무를 순리롭게 완성 했다.

    多亏大家，任务顺利完成了。

  • 덕분에 모두 잘 있어요.

    托您的福，大家都健康。

  • 덕문에 무사합니다.

    托您的福，平安无事。

  • 선생님 덕분에 합격했어요.

    多亏老师才合格了。

# 다기에 / 라기에

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 (ㄴ/는)다 // 체언 (이)라 + 기에

설명  表示说话人听到的传闻成了原因或依据。 = “因为听说……”

주의  终结语尾不能使用命令式或共动式。

예문  

  • TV를 보니까 반신욕을 하면 피부에 좋다기에 나도 해 봤어요.

    听电视里说如果洗半身浴可以让皮肤变好，所以我也试了试。

  • 같이 여행 갈 사람을 찾는다기에 친구를 소개해 주었어요.

    听说他要找一起去旅行的人，所以把朋友介绍给了他。

  • 아들이 잃어버린 돈지갑을 찾았다기에 시름을 놓았어요.

    听说儿子找到了钱包，我就放心了。

  • 듣기 여습에 좋다기에 한국 드라마를 자주 봐요.

    听说对听力有帮助，所以经常看韩剧。

  • 결혼 기냄일이라기에 좀 특별한 것을 선물하고 싶어요.

    因为听说是结婚纪念日，所以想送特别一点的礼物。

  • 요즘은 백화점이 세일 기간이라기에 친구와 같이 백화점에 갔어요.

    听说最近百货商店打折，所以跟朋友一起去了。

# 다길래 / 라길래

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 (ㄴ/는)다 // 체언 (이)라 + 길래

설명  通过引用他人的话来强调理由，原因，常用于口语中。

예문  

  • 배가 고프다길래 라면을 끓여줬어요.

    他说肚子饿，所以给他煮了方便面。

  • 이 영화가 재미 있다길래 보러 갔어요.

    听说这部电影很有意思，所以去看了。

  • 오후부터 비가 온다기래 우산을 가지고 출근했어요.

    听说下午开始下雨，所以带着雨伞上班了。

# 다니까 / 라니까

접속  동사 어간(ㄴ/는) // 형용사 어간 -았 / 었 / 였 / 겠 다 // 체언 (이)라 + 니까

설명  表示前项引用的部分成为后项结果的原因，表示说话人的语气很强。

예문  

  • 그 사람이 하루에 5km씩 걷는다니까 모두들 놀랐어요.

    他说他每天走五公里，大家都很吃惊。

  • 급하다니까 빨리 끝내세요.

    他们说很着急，所以你要快点结束。

  • 오후에 비가 온다니까 우산을 가지고 가세요.

    听说下午下雨，所以带着雨伞出门吧！

# 답시고 / 갑시고

접속  동사 어간 / 형용사 어간 / -았 / 었 / 였 / 겠 (ㄴ/는)답 // 체언 (이)랍 + 시고

설명 I  表示倚仗某种条件或理由而做某事。

예문 I  

  • 사장이랍시고 뻐기고 다니더라.

    仗着自己是社长，在那儿都趾高气扬。

  • 언니는 한 푼이라도 벌겠답시고 취직하러 베이징으로 갔어요.

    姐姐说赚到一分钱也好，就去北京工作了。

  • 너도 좀 배웠답시고 나서는 거냐?

    仗着学了点东西，就站出来显摆吗。

  • 미령은 예쁘답시고 남자들에게 쌀쌀맞게 군다.

    美玲仗着自己漂亮，对那些男的很冷漠。

설명 II  表示本想做某事，但却没有达到预期的效果，带有嘲讽的语气。

예문 II  

  • 친구가 머리를 예쁘게 잘라준답시고 요꼴로 만들었어요.

    朋友说是给我剪一个漂亮的发型，结果搞成这样。

  • 그사람은 그림을 그린답시고 방 안에서 빈동거리기만 했어요.

    那个人说是画画，其实是在房间里游手好闲。

  • 호랑이를 그린답시고 고양이를 그렸다.

    说是画老虎，画成了猫。

# 까닭이다 / 에 / 으로

접속  동사 어간 / 형용사 어간 / 이 (다) + (으)ㄴ/는까닭이다/에/으로

설명  表示某种事情发生的原因、理由。

예문  

  • 아직 나의 청춘이 다 하지 않은 까닭입니다.

    因为我的青春尚未燃尽。

  • 너무 설렜던 까닭일까요?팬미팅 당일 큰 실수를 하고 말았습니다.

    可能是太激动了，粉丝见面会当天我闯了大祸。

  • 날씨가 추운 까닭에 요즈음 밖에 나오는 사람이 적다.

    天气很冷，最近外出的人少。

  • 그가 무슨 까닭으로 그런 말을 했을까요?

    他因什么缘故说了那些话呢？

# 니만큼

접속  동사 어간 / 형용사 어간 / 이 / 았 / 었 / 였 + (으)니만큼

설명  连接词尾。表示原因、根据。强调基于前项条件，应该做某事或理应出现后项情况。= “正因为……应该……”

예문  

  • 부모가 잘 생겼으니만큼 그집의 아이들도 예쁘게 생겼을 거야.

    父母长得好看，他们家的孩子也肯定会很漂亮。

  • 요즘은 장마절이니만큼 제때제때 빨래를 말려야 할 거예요.

    因为这段时间是梅雨季，要及时晾衣服。

  • 사간이 축박하니만큼 일찍부터 서둘러야 해요.

    因为时间很紧，得早点准备了。

  • 학교가 머니만큼 일찍 나가야 한다.

    因为学校远，要早点出发。

  • 그녀의 글씨는 단아하고 청초하고 반흘림이니만큼 읽기 좋았다.

    她的字体端雅清秀，还是半草书体，所以看起来很舒服。
