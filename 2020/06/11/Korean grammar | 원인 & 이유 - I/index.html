Korean grammar | 원인 & 이유 - I

>> 表原因/理由 고 해서 | 기 때문에 | 기에 | 길래 | 느라(고) | 는 바람에 | 는 통에 | 다(가) 보니(까)

# 고 해서

접속  동사 어간 / 형용사 어간 / 이 + 고 해서

설명  表示导致某事发生的多种原因中，只列举其中一个，即除了提到的原因外还有其他原因 常用 “도 - 고해서“

예문  

  • 날씨가 따뜻해지고 해서 아침마다 산책을 가요.

    天也暖和了，所以每天早上都去散步。

  • 주변도 조용하고 해서 그 동네로 이사를 했어요.

    周边很安静，所以就搬到那个小区了。

  • 장학금을 받고 해서 친구들과 한잔했어요.

    得了奖学金，所以和朋友们出去喝了一杯。

  • 기분도 울적하고 해서 산책을 나가기로 했다.

    心情也有点郁闷，所以决定出去走走。

  • 이러다 죽는 거 아닌가 하고 겁도 나고 해서 그를 꼭 껴안았다.

    照这样下去岂不是会死，觉得害怕所以紧紧抱住了他。

# 기 때문에

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 / 겠 + 기 때문에

설명  表示原因 = “因为，所以”

주의  表示原因或理由的内容出现在谓语部分时，往往以기 때문이다的形式结尾，如句4。

예문  

  • 주말이기 때문에 백화점에 사람이 엄청 많아요.

    因为是周末，所以商场的人很多。

  • 은정 씨는 어제 밤새 공부기 때문에 오늘 늦게 일어났어요.

    恩静昨晚熬夜学习，所以今天起来得很晚。

  • 평수에 아주 바쁘기 때문에 친구들과 자주 연락을 못합니다.

    因为平时很忙，所以不能经常和朋友联络。

  • 현대 사회가 하루가 다르게 변화하고 있어 ‘빨리빨리’ 적응하는 것이 성공에 유리하기 때문이다.

    那是因为现代社会发生日新月异的变化，快速适应有利于成功。

# 기에

접속  동사 어간 / 형용사 어간 / 이 / -았 / 었 / 였 + 기에

설명  接续词尾。表示原因或依据 = “因为……”

예문  

  • 반가운 손님이 오셨기에 버선발로 달려 나갔다.

    因为来了贵客，所以连鞋子都没有穿，急忙跑出去迎接客人了。

  • 전번 조말에는 어디로 가셨기에 전회도 안 통해요?

    上周末您去了哪里了，怎么电话打不通？

  • 도대체 얼마나 울었기에 눈이 퉁퉁 부었을까?

    到底哭了多久，眼睛会肿成那样？

  • 학생들이 두대체 무슨 죄를 지었기에 저런 징벌을 받아야 하는가?

    学生们到底犯了什么罪，要受到那样的惩罚？

# 길래

접속  동사 어간 / 형용사 어간 / 이 (다) / -았 / 었 / 였 + 길래

설명  表示原因，常用于口语中 “기에” 的口语型

예문  

  • 배가 너무 고프길래 라면을 끓여 먹었어요.

    肚子太饿了，所以煮了泡面吃。

  • 얼마나 그 사람이 좋길래 그렇게 정신을 못 차리냐?

    那个人是有多好，把你迷得这样神魂颠倒。

  • 아무리 약을 먹어도 안 길래 병원에 기봤다.

    再怎么吃药也没效果，所以去了医院。

# 느라 (고)

접속   동사 어간 + 느라(고)

설명 I  接续词尾。表示原因。前后句子的主语一致，往往省略后句主语。 느라고 可以缩写为 느고

예문 I  

  • 자느라고 도둑이 들어온 걸 몰랐어요.

    睡着了，所以不知道小偷进来了。

  • 공부하느라고 다크서클이 생겼어요.

    因为学习，出现了黑眼圈。

  • 저희 부무님은 농장을 돌보시느라고 바쁘십니다.

    我的父母因为照料农场，所以很忙。

  • 그동안 어머니를 모시느라 고생 많았서.

    一直以来侍奉母亲，辛苦你了。

설명 II   接续词尾。表示目的。前后句子的主语一致，往往省略后句主语。 느라고 可以缩写为 느고

예문 II

  • 은지는 옷음을 참느라고 딴 데를 보았어요.

    恩知为了忍住不笑，看了别的地方。

  • 방을 구하느라 뛰어다녀요.

    为了找房子到处奔走。

  • 그녀는 눈물을 감추느라 한참 눈을 감고 있었다.

    她为了忍住泪水，闭了好一会眼睛。

  • 영화를 보느라 점심을 긂었다.

    为了看电影，没有吃午饭。

# 는 바람에

접속  동사 어간 + 는 바람에

설명  表示因为某事而导致意想不到的结果，这一结果往往是消极的，负面的。

예문  

  • 급히 먹는 바람에 체햤다.

    因为吃太急，所以噎着了。

  • 어제 눈이 오는 바람에 길이 미끄러웠다.

    因为昨天下雪，所以路面很滑。

  • 아이는 배탈이 나는 바람에 학교에 결석했다.

    因为孩子吃坏了肚子，所以没去学校。

# 는 통에

접속  동사 어간 / 형용사 어간 + 는 통에

설명  表示原因或根据，带有很强的责怪或不满的语气。

예문  

  • 아이들이 떠드는 통에 잠을 잘 수가 없어요.

    孩子们太吵了，没法睡觉。

  • 급하게 나오는 통에 휴대폰을 잊고 나왔어요.

    出来得太急，所以出来时候把手机落下了。

  • 담당자가 없는 통에 업무가 영 뒤죽박죽이 뒤고 말았어요.

    因为没有负责人，业务变得乱七八糟。

# 다 (가) 보니 (까)

접속  동사 어간 / 형용사 어간 / 이 + 다(가) 보니(까)

설명  引出结果。表示长期处于某种状态，自然引发出另外一个结果。这是由 “다가 (表示原先动作发生变化或停顿)”和“보니까 (表示原因、理由)”组合而成的句型。“보니까”也可以说成“보니”

예문  

  • 오래 살다 부니까 별꼴을 다 보겠네.

    人活久了，什么事都能遇见。

  • 너랑 즐겁게 놀다 보니까 걱정이 다 날아간 거 같아.

    跟你玩得好开心，我的烦恼好像都消失了。

  • 매일 한국 친구를 만나다 부니까 한국어를 잘 하게 됐어요.

    天天见韩国朋友，自然韩语越来越流利了。

  • 저의 친구들은 다 남자 친구가 있는데 제가 좀 똥똥한 편이다 보니까 님자친구도 없어요.

    我的朋友们都有男朋友，但是我因为胖，还没有男朋友。

  • 연못을 들여다 보니까 과연 물속에 황금의 그림자가 비치고 있었다.

    往莲花池里一看，果然看到水里映出金色的影子。

  • 워낙 바쁘다 보니까 다들 생각하지 못했나 봐요.

    可能是因为大家都很忙，所以没想到吧。

  • 돈을 너무 아끼다 보니 인생을 즐기지 뭇했어요.

    太节省了，结果都没能享受人生。

  • 친구의 재미있는 이야기를 듣다 보니 시간 가는 줄 물랐네요.

    听朋友讲有趣的故事，结果都不知道时间过得这么快。

  • 해외에 나갈 기회가 없다 보니 우물 안 개구리가 되었네요.

    一直没有机会出国，结果都成了井底之蛙。
