NLP lecture 1 | Language & Writing systems

>> Notes for NLP course given by Prof. Lepage

在IPS的第二学期，没有实验室和教室，配置就是自费的宿舍、自己的laptop、自开的热点和一只来自印度的可爱舍友，两节网课加每周的网络seminar就是值5w的一学期，十分美好。不过我这法国人老板的课是真的干货，于是我决定好好做笔记。

Natural Language Processing 的第一节课太有老板的风格，作为一个会数不清种语言的Polyglot, 他一定是有分享不完的语言知识和故事，所以第一节肯定是从语言讲起，上学期的Distributional Semantics也一样，永远绕不开语言是什么，语言是怎样的这个话题。

Languages

看了一些语言学的描述，语言特指人类语言，区别于其他动物之间的信息传递，仅人类独有。所以如果要严谨的描述动物之间的“语言”，这个引号还是不要省略为好。语言最主要的特征应该在于它系统而繁杂的“规则”，也就是语法，正是这样既定的规则让语言能够成为传递无限信息的工具，同时也只有人类的具有这样的认知能力来使用它。

How many languages in the world?

我一直以来也陷入一个误区，觉得好像只有那些能写下来的，有文字的，被当成官方语言的语言才叫语言，其他的顶多叫方言。其实语言学上来说，所有这些能用来交流的，都叫语言，区别在于有些语言有文字，而其它只有口语形式。

世界上目前大概有6000种语言，但也许将会有一千种以上会彻底消失在这个世纪。这些语言中大概200到400具有文字，也就是能被写下来，如汉字、假名。

Language codes

每一种语言在不同地区可能有不一样的名字，为了统一标准，方便在任何情况（尤其是计算机应用和其它语言学研究）我们都能指定一种语言，ISO 639语言代码就产生了。还有一个原因是，如果用国家或地区来命名语言，会造成很多分歧，因为一种语言不只有一个国家的人使用，也不只有一个地区，一个种族的人使用。另外因为语言一直在增减，现在已经从2002年的ISO 639-1起更新到了ISO 639-6。这套代码表用不同2或3个小写字母表示各语言。像我们经常可以在输入法栏看到代表中文的“zh”，以及英文的“en”，日文的“ja”，韩语的“ko”都是语言代码。

Writing System

  • 文字 / 书写系统

  • 文字 ≠ 语言

    • 越南语历史上经历过三种文字：

        汉文（文言文）

        汉喃文（一种改编于汉字的文字）

        国语字（越南罗马字）（17世纪之后）

        >> 越南语介绍

    • 教授在课上提到说拼音当年的出现是为了废除汉字，个人还是非常震惊的，查阅了下相关的资料，更懵的是连我们的鲁迅先生，当年也有”汉字不灭，中国必亡”的论断。老师说拼音的产生是为了消灭汉字其实不太准确，拼音是1957年公布的《〈汉语拼音方案草案〉的决议》正式确立的，目的是为了促进汉字的普及度无可非议。但是在此之前的近代，在经历西方国家的种种霸凌之后的中国，确实掀起了一大波汉字拉丁化的运动。

       >> 如何看待鲁迅的「汉字不灭，中国必亡」？ –知乎

       >> 汉语拼音的由来

  • 文字 ≠ 语系

    • 日语中有汉字，但是日语算是孤立语系，而普通话属于Sino-Tibetan语系

Classification of writing systems

https://zh.wikipedia.org/wiki/%E6%96%87%E5%AD%97

  1. Logographic systems 意指文字

    Logogram（语素文字）两河流域的楔形文字、古埃及的圣书字，中国的汉字以及与汉字相关的其他文字，如越南的喃字，古彝文。

  2. Syllabic system 音节文字

    kana 假名 / cree

  3. Alphabetical systems 音素文字

    • Abjad 辅音音素文字 (只有辅音音素)
      • 辅音音素文字特点是每个符号都代表辅音，辅音音素文字的字母表只有辅音字母，没有元音字母，所以朗读文章时口中须补上适当的元音才能读出。典型的辅音音素文字有“腓尼基字母”、“阿拉伯字母”等。
      • 阿拉伯文字
      • 希伯来字母
      • 提非纳文字
    • Alphabet 全音素文字
      • 全音素文字是表音文字的一种，它是以音素为单位的文字。和不标出元音的辅音音素文字不同，它的字母表中除了辅音字母，还有元音字母，用来表示语言中的元音
      • 拉丁字母
      • 西里尔字母
      • 希腊文
      • 亚美尼亚字母
      • 格鲁吉亚字母
      • 蒙古文字
      • 谚文
      • 它拿字母
      • 古匈牙利字母
    • Abugidas 元音附标文字
      • 以辅音字母为主体、元音以附加符号形式标出的表音文字。该书写系统的主要特点是：辅音字母本身即有元音，一般是/a/，构成一个音节；其它元音字母一般附加在辅音字母的周围（前后上下均有可能），替代默認元音，以改变音节的读音。
      • 天城文
      • 泰语字母
      • 缅甸文
      • 老挝文
      • 高棉文
      • 藏文
      • 印地文
      • 梵文
      • 悉昙文字
      • 泰米尔文
    • 僧伽罗文

  4. Featural system 特征文字

    • Hangul 谚文

  5. Ambiguous systems

    • Bopomofo 注音符号（台湾的“拼音”）（可以说是一种Alphabet）
    • 包括其它一些文字，包含了很多不同种类的文字而很难被分类。

Segments in writing system

  • Syllables 以音节分，如越南语
  • Words 以词分，如英语
  • Hyperwords or chunks 以词组分，如阿拉伯语，韩语？
  • Sentences or paragraphs 以句子或段落分，如泰语

Conclusion

介绍了这么多，其实是想说语言体系如此庞大，文字系统这么繁杂，从而形成了NLP这样的一个的研究方向，而NLP工具、任务、模型和技术需要经过不同的调整去适应不同的文字系统。但文字系统其实也只是语言的工具，各个语言之间互相借用、演化，而语言学科其实也就是研究其中的规律。
